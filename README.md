# Xamarin-Mac-UI-Samples

## C# Xamarin Mac. 

Xamarin allow me to create UI controls with behavior which I need.
In this repository you will find deferent controls and how you can interact with them. Here is a little list of controlls which I intereact: NSButton, NSTextField, NSTextView, NSplitView, NSTableView, NSNotificationCenter and other UI controls. 

In Util folder you will find little useful examples of code like fingerprint service, native loger, how was with clipboard and other.
